== README

<b>Welcome to MyHome Bookmarks!</b>

<b>A great web application for managing bookmarks! :)</b>
 
Are you tired of using the brower built-in bookmarks?
Do you want to use your bookmarks on different computers and different browsers?

<b>This application is just for you!</b>

Sign up, define your bookmarks, set this page as your home page and use them wherever you need!

This repository contains a Ruby on Rails project used to store user bookmarks online.
